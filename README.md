# React Todo App

리액트를 기반으로 만든 todo 앱입니다.

## 시작하기

`npm start`   
이 앱을 개발모드로 실행시켜줍니다.<br>
[http://localhost:3000](http://localhost:3000) 로 브라우저로 열어 볼 수 있습니다.

이 페이지는 변경사항이 있을때마다 수정됩니다.<br>

`npm run build`   
배포를 위한 앱이 `build` 폴더에 생성됩니다.<br>

### 개발전제조건
이 앱은 노드를 사용하여 react와 sass를 이용하고 있습니다.   
```npm i```
